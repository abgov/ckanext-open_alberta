For Developers

The Government of Alberta is committed to publishing open data to third parties via data feeds through an application programming interface (API). The Government of Alberta encourages developers to deliver and innovate using open data.

The base URL for the API is: http://open.alberta.ca/api.

The API is provided by the CKAN software and full instructions are provided here: http://docs.ckan.org/en/latest/api/index.html.

Questions about how you can use our information? Learn more in the licence section.
